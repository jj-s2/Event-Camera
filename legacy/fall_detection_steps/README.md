# Legacy Fall-Detection Step Scripts

This directory preserves the original `Step_*.py` workflow that was present
before the event-camera industrial defect project was added.

These scripts are kept for traceability, but the maintained project entry
points now live in:

- `event_defect/` for reusable Python package code
- `scripts/` for dataset conversion, training, evaluation and demos
- `docs/` for algorithm, data and experiment notes

No behavior was intentionally changed while moving these files.
